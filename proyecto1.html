Sobre el Proyecto

Un panel interactivo (Single Page Application) diseñado para transformar la preparación del examen AWS Certified Cloud Practitioner.

Este proyecto no es solo un resumen de conceptos; es una herramienta diseñada para integrarse en un ecosistema de aprendizaje moderno donde la Inteligencia Artificial actúa como tu tutor personal. La memorización de manuales técnicos extensos ha quedado atrás. Este panel estructura la información tal y como la procesaría un sistema de aprendizaje optimizado.

El Paradigma de Aprendizaje

  • Definiciones precisas: Lo estrictamente necesario sin ruido de fondo.
  • Analogías del mundo real: Para generar anclajes mentales fuertes y rápidos en conceptos complejos de la nube.
  • Datos Clave (Snippets): La información exacta que los algoritmos de evaluación de AWS suelen utilizar en sus preguntas trampa.

Integración con Gemas de Gemini

Para cerrar el ciclo de aprendizaje (Teoría → Práctica → Validación), este proyecto se apoya en el uso de Gemas de Gemini.

En lugar de buscar exámenes estáticos, utilizas una IA configurada como tu "Examinador AWS". La IA evalúa tus respuestas y te explica el razonamiento arquitectónico detrás de cada fallo.

Arquitectura y Tecnologías

El proyecto es totalmente Serverless, no requiere instalación de dependencias y está diseñado bajo el concepto de Single Page Application (SPA) para garantizar una carga instantánea y accesibilidad visual.

  • HTML5 Semántico
  • CSS3 (Variables, Flexbox)
  • Vanilla JavaScript (ES6+)
  • Serverless (GitHub Pages)
