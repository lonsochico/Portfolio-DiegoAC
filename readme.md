# 🚀 Construyendo la Identidad Digital: IT Portfolio & Project Hub

**Contacto:** [LinkedIn](https://www.linkedin.com/in/diego-alonso-chico-1301a4183) | [GitHub](https://github.com/lonsochico)

Este repositorio contiene el código fuente de mi portfolio profesional.  
En el sector tecnológico actual, un currículum estático ya no es suficiente.  
Este proyecto nace de la necesidad de poseer una herramienta dinámica y centralizada que no solo enumere mis habilidades, sino que demuestre de forma tangible mi capacidad para resolver problemas complejos y ejecutar proyectos de gran envergadura.
Un portfolio es mucho más que una web personal; es el puente entre el conocimiento técnico y la ejecución estratégica. He desarrollado esta plataforma con tres objetivos fundamentales:

- **Visibilidad de Metas:** Proyectar mi evolución desde la administración de sistemas tradicional hacia arquitecturas Cloud e IA Responsable.
- **Narrativa de Proyectos:** Explicar el "cómo" y el "porqué" de mis decisiones técnicas, desde despliegues críticos en aeropuertos internacionales hasta la automatización de infraestructura global.
- **Sinergia de Habilidades:** Mostrar un perfil híbrido que combina la solvencia técnica (Hard Skills) con el liderazgo y la excelencia operativa aprendida en el sector servicios de lujo (Soft Skills).

## 🛠️ Arquitectura y Tecnologías

El portfolio ha sido construido bajo los principios de velocidad de carga, diseño responsivo y estética corporativa moderna.

- **Frontend:** HTML5 semántico, CSS3 (Variables, Grid, Flexbox) y Vanilla JavaScript (ES6+).
- **Diseño:** UI inspirada en centros de mando de ciberseguridad, optimizada para la legibilidad y la navegación fluida.
- **Metodología IA-Assisted:** El diseño y la optimización del código se han llevado a cabo mediante un flujo de trabajo asistido por IA Generativa, demostrando eficiencia en el ciclo de desarrollo.

## 📁 Proyectos Destacados

Cada sección del portfolio profundiza en un hito de mi carrera:

- **Administración Global de Endpoints (TFG):** Diseño de una infraestructura híbrida UEM para una multinacional financiera, gestionando 4.500 dispositivos en más de 50 países con una disponibilidad del 99,99%.
- **Despliegue London Heathrow:** Caso de éxito sobre la migración crítica de infraestructura IT en una ventana nocturna de 10 horas para el líder mundial en cambio de divisas.
- **AWS & IA Didáctica:** Un explorador interactivo diseñado para transformar la preparación de certificaciones Cloud mediante el uso de Gemas de Gemini como tutores personales.
- **Ecosistema Digital "El Abigero":** Despliegue de una solución comercial completa utilizando arquitecturas Serverless y estrategias de conversión directa.

## 📈 Roadmap y Futuro

Este portfolio es un ente vivo. Mi hoja de ruta profesional, integrada visualmente en la plataforma, incluye:

- Certificación AWS Solution Architect Associate.
- Profundización en modelos Zero Trust y políticas de seguridad avanzada.
- Implementación de IA generativa para la automatización de procesos Helpdesk de nivel 2 y 3.
